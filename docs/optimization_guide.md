## 1. 当前性能问题

### 1.1 性能数据

| 指标 | 当前值 |
|------|--------|
| 处理速度 | 2.96s/张 |
| 测试集规模 | 6,559 张 |
| 预计耗时 | **~5.4 小时** |

### 1.2 瓶颈分析

```
单张图片处理流程:
┌─────────────────────────────────────────────────────────────┐
│  滑动窗口遍历 (60%)                                          │
│  ├── 逐窗口提取 patch                                        │
│  └── 逐窗口计算特征 (35维)                                   │
│                                                              │
│  特征提取 (30%)                                              │
│  ├── DCT (OpenCV 单线程)                                     │
│  ├── ELA (JPEG 编解码)                                       │
│  ├── Noise/Edge/Texture (OpenCV 卷积)                        │
│  └── Color (OpenCV 颜色空间转换)                             │
│                                                              │
│  模型推理 (10%)                                              │
│  └── Random Forest CPU 推理                                  │
└─────────────────────────────────────────────────────────────┘
```

### 1.3 核心问题

| 问题 | 原因 | 影响 |
|------|------|------|
| **滑动窗口低效** | 逐窗口串行处理 | 60% 时间浪费 |
| **特征提取串行** | OpenCV 单线程 | 30% 时间浪费 |
| **GPU 未利用** | Random Forest CPU | 10% 时间浪费 |
| **内存拷贝多** | Python 循环 + numpy | 额外开销 |

## 3. 优化方案

### 3.1 方案对比

| 方案 | 预期加速 | 难度 | 风险 |
|------|----------|------|------|
| **多进程并行** | 2-4x | 低 | 低 |
| **GPU 特征提取** | 10-20x | 中 | 低 |
| **LightGBM 替代 RF** | 1.5-2x | 低 | 低 |
| **综合优化** | **20-50x** | 中 | 低 |
| **深度学习 U-Net** | **100x+** | 高 | 中 |

### 3.2 推荐方案: 综合优化 (已实现)

已创建优化脚本 `detect_fast.py`，包含：

1. **GPU 批量特征提取**
   - 使用 PyTorch 将特征计算移至 GPU
   - 批量处理 256 个 patch，减少内存拷贝

2. **多进程并行**
   - 使用 `multiprocessing.Pool` 并行处理多张图片
   - 自动分配 GPU 设备

3. **优化特征计算**
   - 用 FFT 近似 DCT（更快）
   - 预计算卷积核，避免重复创建

## 4. 使用指南

### 4.1 快速开始

```bash
# 使用优化版脚本
cd /root/forgery

python scripts/detect_fast.py \
    --model ./scripts/results/model \
    --data-dir /ai_paas/limk/workspace/image_tamper/tamper_data_processed \
    --split test \
    --device cuda:0 \
    --num-workers 4
```

### 4.2 参数说明

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `--model` | ./results/model | 模型目录 |
| `--data-dir` | 必填 | 数据目录 |
| `--split` | test | 数据集划分 |
| `--device` | cuda:0 | GPU 设备 |
| `--num-workers` | 4 | 多进程数量 |

### 4.3 多 GPU 并行

```bash
# 使用 GPU 0-3，每个 GPU 2 个进程
CUDA_VISIBLE_DEVICES=0,1,2,3 python scripts/detect_fast.py \
    --num-workers 8 \
    ...
```

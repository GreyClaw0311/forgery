### 1.2 评估指标说明

| 指标 | 公式 | 说明 |
|------|------|------|
| **IoU** | TP / (TP + FP + FN) | 预测掩码与真实掩码的重叠度 |
| **Precision** | TP / (TP + FP) | 预测篡改区域中真正篡改的比例 |
| **Recall** | TP / (TP + FN) | 真实篡改区域被正确检测的比例 |
| **F1-score** | 2×P×R / (P+R) | Precision和Recall的调和平均 |

## 五、结论与建议

### 5.1 当前结论

1. **ELA检测器效果最好**，但仍需大幅优化
2. **主要问题是误报率过高**（Precision < 10%）
3. **单一检测器效果有限**，需要多方法融合
4. **CopyMove检测器需针对性调参**

### 5.2 阶段三优化建议

| 优化方向 | 具体措施 | 预期效果 |
|---------|---------|---------|
| **阈值优化** | 提高阈值，降低误报 | Precision ↑ |
| **后处理** | 形态学操作，去除小区域 | IoU ↑ |
| **多方法融合** | ELA + DCT 融合 | 综合提升 |
| **自适应阈值** | 根据图像特征动态调整 | 鲁棒性 ↑ |

### 5.3 目标

| 指标 | 当前最优 | 目标 |
|------|---------|------|
| IoU | 0.0735 | ≥ 0.30 |
| Precision | 7.70% | ≥ 50% |
| Recall | 65.18% | ≥ 60% |
| F1 | 0.1119 | ≥ 0.45 |

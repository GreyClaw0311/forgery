## 一、产品介绍

### 1.1 产品简介

Forgrey 图像篡改检测系统基于多算法融合架构，提供图像篡改检测能力。系统支持四种检测算法（ELA、DCT、Fusion、ML），可精准判断图片是否篡改并定位篡改区域。

### 1.2 核心特点

- ✅ **两阶段检测**: GB分类器前置判断 + 区域定位算法
- ✅ **多算法支持**: ELA、DCT、Fusion、ML 四种检测算法
- ✅ **像素级检测**: ML 算法支持像素级篡改区域定位
- ✅ **篡改区域坐标**: 输出篡改区域矩形坐标
- ✅ **GPU 加速**: 支持 PyTorch GPU 加速推理
- ✅ **REST API**: FastAPI 服务，易于集成

### 1.3 应用场景

| 场景 | 描述 |
|------|------|
| 政务文档审核 | 检测公文、证件等文档是否被篡改 |
| 司法取证 | 提供图像真实性鉴定技术支持 |
| 金融风控 | 检测合同、票据等金融文档真伪 |
| 新闻媒体 | 核实新闻图片真实性，防止虚假传播 |
| 电商平台 | 检测商品图片是否经过不当编辑 |

### 1.4 产品优势

| 优势 | 说明 |
|------|------|
| 多算法融合 | 四种算法互补，覆盖多种篡改类型 |
| 高准确率 | ML算法F1可达XX%以上 |
| 低资源消耗 | CPU即可运行，GPU可选加速 |
| 易于部署 | Docker容器化，一键部署 |
| API接口 | 标准REST API，易于集成 |

## 二、算法概览

### 2.1 算法对比

| 算法 | 全称 | 原理 | 输出类型 | 适用场景 |
|------|------|------|----------|----------|
| **ELA** | Error Level Analysis | JPEG压缩误差分析 | 热力图 | JPEG图像篡改检测 |
| **DCT** | Discrete Cosine Transform | DCT块效应检测 | 热力图 | 块效应篡改检测 |
| **Fusion** | Multi-feature Fusion | 多特征融合检测 | 热力图+Mask | 综合篡改检测 |
| **ML** | Machine Learning | 像素级机器学习 | 精确Mask | 高精度篡改定位 |

### 2.2 检测流程

```
输入图片 → GB分类器(前置判断) → 选择算法 → 区域定位 → 输出结果
                                    ↓
                              ┌─────────┐
                              │   ELA   │ JPEG压缩误差
                              │   DCT   │ DCT块效应
                              │ Fusion  │ 多特征融合
                              │   ML    │ 像素级检测
                              └─────────┘
```

## 七、API接口规范

### 7.1 接口信息

| 字段 | 内容 |
|------|------|
| 接口协议 | HTTP REST API |
| 接口地址 | http://host:8000/detect |
| 请求方法 | POST |
| Content-Type | application/json |

### 7.2 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| image_base64 | string | 是 | 图片Base64编码 |
| algorithm | string | 是 | 检测算法: ela/dct/fusion/ml |
| return_mask | boolean | 否 | 是否返回Mask图片 (默认true) |
| return_marked | boolean | 否 | 是否返回标注图片 (默认true) |

### 7.3 响应参数

| 参数 | 类型 | 说明 |
|------|------|------|
| code | string | 状态码 (0001=成功) |
| message | string | 状态信息 |
| data | object | 检测结果 |
| data.is_tampered | boolean | 是否篡改 |
| data.confidence | float | 置信度 (0-1) |
| data.algorithm | string | 使用的算法 |
| data.mask_base64 | string | 检测结果Mask (Base64) |
| data.marked_image_base64 | string | 标注图片 (Base64) |
| data.tamper_regions | array | 篡改区域坐标列表 |
| data.inference_time | float | 推理耗时(秒) |

### 7.4 状态码说明

| 状态码 | 描述 |
|--------|------|
| 0000 | 未知异常 |
| 0001 | 处理成功 |
| 0002 | Base64解码异常 |
| 0004 | 参数格式错误 |
| 0007 | 参数为空或无法解析图片 |

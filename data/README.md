# 测试数据说明

本目录存放测试数据，仅包含少量测试样本。

## 目录结构

```
data/
├── easy/images/      # 简单篡改图片
├── difficult/images/ # 复杂篡改图片
└── good/             # 正常图片
```

## 数据说明

| 类别 | 描述 | 建议数量 |
|------|------|----------|
| easy | 简单篡改图像 | ≥ 20张 |
| difficult | 复杂篡改图像 | ≥ 16张 |
| good | 正常未篡改图像 | ≥ 10张 |

## 数据格式

- 支持格式: JPG, PNG, BMP
- 建议分辨率: 不限

## 获取全量数据

全量数据集请参考项目文档或联系作者获取。

## 注意事项

- 测试数据仅供功能验证
- 训练模型建议使用更多数据
- 数据集需遵守相关版权规定
